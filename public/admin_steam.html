Авторизация Бота

Введите данные для получения API сессии. Пароли не хранятся в открытом виде.

Как получить куки:
1. Зайди в Steam через браузер (Chrome/Safari).
2. Авторизуйся в аккаунте бота.
3. Нажми F12 -> Application -> Cookies -> steamcommunity.com.
4. Скопируй значения sessionid и steamLoginSecure.
